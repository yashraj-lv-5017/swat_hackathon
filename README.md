# 🛡️ Axion Brand Sentinel
**Enterprise Brand Governance & Content Adaptation Engine**

## 🚀 Overview
The Sentinel uses **RAG (Retrieval-Augmented Generation)** to audit marketing content against Axion’s core brand laws stored in a **Qdrant Vector Database**. Powered by **GPT-5.4-beta**, it adapts any input text for 9 different permutations of Audience and Channel.

## 🛠️ Tech Stack
- **Model:** Azure OpenAI GPT-5.4-beta
- **Vector DB:** Qdrant (Dockerized)
- **UI:** Streamlit
- **Logic:** Python RAG (Semantic Search + Persona Injection)

## 🎯 Features
- **Brand Health Scoring:** Real-time compliance metrics.
- **9x9 Permutation Logic:** Instant pivot between LinkedIn, Email, and PR for Executives to Tech Leads.
- **Automated Audit Trail:** Detailed change logs with rule citations (§).
